# GithubKeyboardCheatsheet

|Keyboard|Task|
|---------|--------|
|<kbd>,</kbd> |Opens IDE Codespaces (where you can edit and work with repo in full vs environment,including terminal etc)|
|<kbd>.</kbd> |Opens github.dev (where you can edit code only )|
|<kbd>G</kbd> + <kbd>C</kbd> |Opens Code tab under repo section|
|<kbd>G</kbd> + <kbd>P</kbd> |Opens PR tab under repo section|
|<kbd>G</kbd> + <kbd>I</kbd> |Opens Issues tab under repo section|
|<kbd>G</kbd> + <kbd>A</kbd> |Opens Actions tab under repo section|
